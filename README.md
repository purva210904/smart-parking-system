# Smart Parking System

Embedded system for vehicle detection and parking space monitoring.

## Features

* Vehicle detection
* Parking slot monitoring
* Sensor-based automation
* Real-time indication

## Components Used

* Arduino UNO
* IR Sensors
* LEDs

## Technologies Used

* Arduino IDE
* Embedded C
* Sensor Interfacing

## Applications

* Smart parking systems
* Traffic management
* Automated vehicle monitoring
